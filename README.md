Руководство пользователя:
1) Скачать Arduino IDE и установить.
2) Открыть Arduino IDE: Sketch → Include library → установить:
- DHT sensor library 
- LiquidCrystal 
3) Cобрать аппаратную часть (схемы в папке "Схемы")
4) Подключить Arduino по USB.
5) В Arduino IDE: Инструменты → Плата → Arduino Uno
6) Запустить код (код в папке "Код")
7) После загрузки на LCD-экране отобразятся температура (°C) и влажность (%) в реальном времени.
